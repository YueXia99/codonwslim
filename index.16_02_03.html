Correspondence Analysis of Codon Usage

CodonW is a programme designed to simplify the Multivariate analysis (correspondence analysis) of codon and amino acid usage. It also calculates standard indices of codon usge. It has both menu and command-line interfaces. It was written by John Peden as part of his PhD project in the lab of Paul Sharp, CMELUN, Dept of Genetics, University of Nottingham. John is currently employed at the MRC Haematology Unit at the IMM who are kindly hosting this page.

NB: I have extensively tested it and believe it to be bug free. If you find a bug or something confusing please email me at John.Peden@molbiol.ox.ac.uk and let me know (Don't forget to tell me the Operating System and to send me the input file that generated the error).

Downloading and Installation

Binaries are available for the following operating systems. Alternatively you can download the source code in tar, compressed or gzipped format and compile it yourself. See the README file for more information.

1. Digital UNIX 4.0

2. Irix 6.5 (64 bit)

3. Solaris 2.6 (solaris)

4. Win 32bit (NT/95/98/2000) (No GUI; runs within a DOS window)

5.(Sorry no Mac version the source code should compile on CodeWarrior but I don't have a MAC compiler, bewarned it doesn't have any GUI)

6. Web access via PISE at the pasteur http://bioweb.pasteur.fr/seqanal/interfaces/codonw.html

Features:

  1. Written in ANSI compliant C
  2. Menu driven
  3. Extensive number of command line options
  4. Genetic code independent 
  5. No limits on 
    1. number of sequences 
    2. sequence length
  6. Calculates the codon usage indices
      7.  
    1. CAI: Codon adaptation index
    2. Fop: Frequency of optimal codons
    3. Nc: Effective number of codons
    4. CBI: Codon Bias Index 
  7. Calculates amino acid indices
      8.  
    1. GRAVY score
    2. Aromaticity
  8. Calculates correspondence analysis of 
    1. Codon Usage
    2. RSCU (Relative synonymous codon usage)
    3. Amino acid usage
  9. Correspondence analysis 
    1. can include/exclude codons/amino acids
    2. can generate detailed reports of trends
    3. attempts to identify optimal codons automatically
    4. can allow additional data sets to be added 
    5. records any number of trends
  10. Calculates gene parameters
      11.  
    1. Gene length
    2. GC, GC3s and codon position specific G+C 
    3. Dinucleotide composition (in all three codon frames)
    4. Amino acid usage
    5. Relative amino acid usage
    6. Codon usage
    7. Relative Synonymous codon usage
  11. Can conceptually translates sequences to protein
  12. Reformats sequence data
  13. Human or machine readable output
  14. Can concatenate genes (whilst preserving reading frame) to calculate
      15.  
    1. overall codon usage
    2. GC content
    3. Dinucleotide composition 
  15. Generates tables of codon, amino acid, or RSCU usage
  16. Can even help you learn the Genetic code.
  17. and more
